## Description

Allow the user to copy conversion results to the clipboard with a key press.

## Requirements

- Press "c" to copy the current conversion results to the clipboard
- Show a brief confirmation message when copied
- Copy the raw value (without formatting/styling)

## Testing

- Enter a value, press "c", paste elsewhere to verify
- Verify confirmation message appears and disappears
